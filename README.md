## Instructions

### Clone repository:

```bash
git clone git@github.com:irina-tim/todo-list.git
```

### cd to project directory and run:

install dependencies

```bash
npm install
```

serve with hot reload at localhost:8080
```bash
npm run dev
```

build for production with minification
```bash
npm run build
```
